Differences Between HTML and HTML5

HTML HTML5
HTML stands for "Hypertext Markup Language." HTML5 is the fifth version of HTML, which stands for "Hypertext Markup Language 5."
HTML4, the predecessor, was less feature-rich and lacked some modern web capabilities. HTML5 introduced new features, including native support for audio, video, and improved semantic elements.
HTML4 didn't support features like the `canvas` element for graphics or the `video` and `audio` elements for multimedia. HTML5 added support for the `canvas` element for creating graphics and introduced the `video` and `audio` elements for embedding multimedia.
