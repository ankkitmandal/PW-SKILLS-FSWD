Full Stack Web Development Tech Stack:

HTML
(Hypertext Markup Language ),the fundamental markup language used to structure web content.
CSS
Cascading Style Sheets, used for styling and layout, CSS controls the visual presentation of web pages.
JavaScript
A versatile scripting language that adds interactivity and functionality to web pages. Popular libraries and frameworks include:React.js, Angular, Vue.js
MongoDB
A flexible NoSQL database for storing application data
Express.js
A back-end framework for building APIs and handling server-side logic.
React
A front-end library for creating interactive user interfaces.
Node.js
A runtime environment that allows server-side JavaScript execution.
