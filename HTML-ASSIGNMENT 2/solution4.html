HTML Tags

  1. h1: Defines the biggest heading of the web page
  2. p: Defines a paragraph
  3. ol: Define Ordered list
  4. ul: Defines unordered list
  5. hr: Defines horizontal line
  6. br: Defines Line break
  7. a: Defines Hyperlink
  8. table: Defines table with row and column
  9. div: Defines a container for HTML
  10. i: Defines Italic annotation for text
